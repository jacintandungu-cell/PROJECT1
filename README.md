Wordly Dictionary is a *Single Page Application (SPA)* built with HTML, CSS, and JavaScript.  
It allows users to search for words, view their definitions, synonyms, and pronunciation, and save favorite words for future reference — all without refreshing the page.  

This project demonstrates proficiency in:
- DOM manipulation
- Event handling
- Fetch API usage
- Error handling
- LocalStorage integration
- Responsive design and user experience

---

##  Features
-**Search Functionality**: Enter a word and fetch its definition from the Free Dictionary API.
- **Data Display**: Shows part of speech, definition, example usage, synonyms, and phonetic spelling.
- **Audio Pronunciation**: Plays audio if available.
- **Favorites**: Save words to localStorage and view them in a dynamic list.
- **Error Handling**: Displays clear messages for invalid inputs or missing data.
- **Polished UI**: Responsive card layout, gradient background, hover effects, and accessible design.
